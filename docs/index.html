소개

SessionMeter는 Claude와 Gemini의 사용량을 시스템 트레이 아이콘, 데스크톱 위젯, 통계 창으로 보여 주는 데스크톱 애플리케이션입니다. 로그인한 서비스마다 5시간·주간 등의 남은 사용량과 초기화까지 남은 시간을 한눈에 확인할 수 있습니다. Tauri v2로 제작된 오픈소스(MIT) 프로그램입니다.

비공식 도구이며 이용약관에 유의해야 합니다. SessionMeter는 로그인된 본인 세션을 이용해 각 서비스의 사용량 페이지와 동일한 데이터를 가져옵니다. 공식/공개 API가 아니며 해당 서비스의 API 키를 사용하지 않습니다. 각 서비스의 약관은 봇·스크립트 등 자동 수단의 접근을 제한하므로 이 방식은 약관에 어긋날 수 있고 계정 제재 위험이 있습니다. 반드시 Anthropic 약관과 Google 약관을 확인하고 본인 책임으로 사용하십시오. 본 프로그램은 Anthropic·Google과 무관하며, "Claude"는 Anthropic, PBC의, "Gemini"는 Google LLC의 상표입니다.

설치

릴리스 페이지에서 Windows 설치 프로그램(SessionMeter_x.y.z_x64-setup.exe)을 내려받아 실행합니다. Windows 10/11에서 동작하며 WebView2 런타임이 필요합니다(최신 Windows에는 기본 포함). 업데이트는 새 설치 프로그램을 실행하면 기존 설정과 로그인 세션이 유지된 채 덮어 설치됩니다. 앱 안에서 새 버전이 확인되면 위젯과 트레이 메뉴에 설치 버튼이 나타나며, 눌러서 바로 업데이트할 수 있습니다.

서비스 로그인

트레이 아이콘을 우클릭해 설정을 열면 계정 목록에서 서비스별로 로그인·로그아웃할 수 있습니다. 로그인한 서비스만 트레이 툴팁·위젯·통계에 나타납니다.

Claude

앱 안에서 claude.ai 로그인 창이 열리며, 로그인을 완료하면 세션이 자동으로 저장됩니다. 세션은 OS 애플리케이션 데이터 폴더의 사용자별 파일에 저장되고, Windows에서는 DPAPI(사용자 범위)로 암호화되어 다른 사용자·오프라인 디스크·복사된 백업에서 복호화할 수 없습니다.

Gemini

Gemini 로그인은 별도의 프로세스로 격리된 창에서 진행됩니다. 이렇게 하면 Google 로그인 페이지에 문제가 있어도 앱 본체(트레이·위젯·다른 서비스)는 영향을 받지 않습니다. 창에서 본인 Google 계정으로 로그인하면 창이 자동으로 닫히고 사용량이 표시됩니다.

Gemini는 공식 API가 없어 gemini.google.com의 사용량 표시를 그대로 읽어오는 실험적 기능입니다. Google이 임베드 브라우저 로그인을 정책적으로 제한하므로, 로그인이 실패하거나 페이지 구조 변경으로 수치가 달라질 수 있습니다.

트레이 아이콘 읽는 법

트레이 아이콘 위에 마우스를 올리면 로그인한 각 서비스의 이름과 남은 사용량, 초기화까지 남은 시간이 툴팁으로 표시됩니다. 남은 사용량은 다음 색상 기준으로 해석합니다.

상태의미(남은 사용량 기준)
여유(녹색)50% 이상 남음
주의(노랑)20% ~ 50% 남음
경고(빨강)20% 미만 남음

사용량은 설정한 갱신 주기마다 자동으로 갱신되며, 설정에서 표시 기준(남은 사용량 / 사용한 사용량)을 바꿀 수 있습니다.

트레이 조작

동작결과
좌클릭위젯 표시 / 숨김
더블클릭통계 창 열기
우클릭메뉴 열기

우클릭 메뉴에는 위젯 스타일, 통계, 설정, 종료가 있습니다. 위젯 표시/숨김은 위젯 스타일 창에서 서비스별로 켜고 끕니다. 메뉴는 앱 테마에 맞춰 다크/라이트로 표시됩니다.

데스크톱 위젯

위젯은 테두리가 없고 작업 표시줄에 나타나지 않는 떠 있는 창으로, 각 항목의 남은 사용량과 초기화까지 남은 시간을 보여 줍니다. 표시 내용에 맞춰 창 크기가 자동으로 맞춰지며, 로그인한 서비스마다 독립된 위젯 창이 뜨고 제목표시줄에 서비스 이름이 표시됩니다.

이번 세션 62%
주간 세션 39%
이번 세션
사용량 62%
초기화 남은 시간 2시간 14분
주간 세션
사용량 39%
초기화 남은 시간 3일 6시간
  • 통계·설정·스타일 바로가기: 상단 아이콘으로 통계 창, 설정 창, 위젯 스타일 창을 엽니다. 폭이 좁으면 아이콘이 점 세 개 메뉴로 접힙니다.
  • 항상 위 고정: 핀 아이콘으로 켜고 끕니다. 켜면 다른 창 위에 항상 표시됩니다.
  • 위치 잠금: 자물쇠 아이콘으로 켜고 끕니다. 잠금 해제 상태에서 위젯을 끌어 원하는 위치로 옮길 수 있으며, 옮긴 위치는 재부팅 후에도 복원됩니다.
  • 투명도: 위젯 스타일 창의 투명도 슬라이더로 조절합니다.

위젯 스타일

트레이 메뉴 또는 위젯 상단의 팔레트 아이콘으로 위젯 스타일 창을 엽니다. 다섯 가지 그래픽 컨셉을 각각 상세·컴팩트 두 형태로, 총 10종 중에서 서비스별로 선택합니다. 실시간 미리보기와 표시 토글(남은 사용량 / 사용한 사용량)을 제공합니다.

위젯의 두 지표 색상은 서비스에 맞춰 표시됩니다(예: Claude 코럴, Gemini 블루). 아래 갤러리는 앱의 실제 위젯 컴포넌트를 그대로 렌더한 것으로(스크린샷 아님, Claude 색상 · 남은 사용량 표시 기준 · 예시 값), 확대해도 선명합니다.

통계 창

트레이 아이콘을 더블클릭하면 통계 창이 열립니다. 로그인한 서비스가 여러 개면 상단 탭으로 구분됩니다. 각 탭에서 다음을 제공합니다.

  • 모든 사용 항목의 남은 사용량, 사용률, 초기화까지 남은 시간
  • 최근 사용 추세 그래프(24시간 / 7일 / 1개월 전환)
  • 현재 속도 기준 소진 예측
  • 항목별 초기화 시각 일정

알림

사용량이 설정한 기준(기본 80%·95%)에 도달하거나 세션이 초기화될 때 데스크톱 알림을 보냅니다. 설정에서 이번 세션·주간 세션 각각의 기준과 초기화 알림을 켜고 끌 수 있습니다.

설치본에서는 알림 발신 주체가 SessionMeter로 표시됩니다. 개발 모드에서는 Windows 특성상 발신 주체가 다르게 보일 수 있습니다.

설정

  • 테마: 시스템 / 라이트 / 다크. 트레이 아이콘, 위젯, 통계, 메뉴에 모두 적용되며, 시스템 선택 시 OS 라이트/다크 전환을 실시간으로 따릅니다.
  • 언어: 자동(OS 언어) / 한국어 / English.
  • 갱신 주기: 1 / 5 / 10 / 15 / 30분.
  • 시작 시 실행: 로그인 시 자동 실행 등록.
  • 알림: 사용량 기준·초기화 알림.
  • 계정: 서비스별 로그인 상태(이름·이메일)와 로그인/로그아웃 버튼.

위젯 관련 설정(스타일, 표시 기준, 투명도, 항상 위, 이동 잠금)은 위젯 스타일 창에서 서비스별로 관리합니다.

문제 해결

  • 사용량이 표시되지 않음: 설정에서 해당 서비스의 로그인 상태를 확인하고, 필요하면 다시 로그인합니다.
  • 세션 만료: 서비스 세션이 만료되면 다시 로그인해야 합니다.
  • Gemini 로그인 화면이 뜨지 않거나 "안전하지 않은 브라우저" 안내가 나옴: Google이 임베드 로그인을 제한하는 경우입니다. 창을 닫고 잠시 후 다시 시도합니다. 로그인 창은 별도 프로세스라 실패해도 앱 본체는 계속 동작합니다.
  • 트레이 아이콘이 보이지 않음: Windows 작업 표시줄의 숨겨진 아이콘 영역(^)을 확인합니다.
  • 수치가 갱신되지 않음: 설정의 갱신 주기를 확인합니다. 갱신은 주기마다 자동으로 이루어집니다.
